# unstorage MMKV

A MMKV driver for unstorage.

## Installation

Install this package as a dependency in the project:

```sh
# npm
npm i unstorage-mmkv

# Yarn
yarn add unstorage-mmkv

# pnpm
pnpm add unstorage-mmkv

# Deno
deno add npm:unstorage-mmkv

# Bun
bun add unstorage-mmkv
```
